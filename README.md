# ⌨️ Typing Speed Test

A simple Typing Speed Test application built using Python and Tkinter.  
It helps users improve typing speed, accuracy, and consistency.

---

## 🚀 Features
- Real-time WPM (Words Per Minute) calculation  
- Accuracy tracking  
- Timer-based test (60 seconds)  
- Random sentence generation  
- Light & Dark theme support  
- Instant feedback (Low, Average, Good, Excellent)

---

## 🛠️ Technologies Used
- Python  
- Tkinter (GUI)  
- Random module
